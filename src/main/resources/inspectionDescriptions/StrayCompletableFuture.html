Reports stray (uncollected) CompletableFuture calls: an expression that produces a java.util.concurrent.CompletableFuture whose value is discarded as an expression statement.

When a future is neither awaited, chained (thenApply, whenComplete, …), returned, nor stored, any exception it completes with is silently swallowed and the caller never learns whether the asynchronous work finished. This is a frequent source of lost errors and races.

Flagged:


CompletableFuture.runAsync(this::work);   // result discarded
service.fetchAsync(id);                   // returns CompletableFuture<?>, discarded
future.thenApply(x -> x + 1);             // new future discarded

Not flagged (the future is collected):


CompletableFuture<Void> f = CompletableFuture.runAsync(this::work);
return service.fetchAsync(id);
allFutures.add(service.fetchAsync(id));
service.fetchAsync(id).join();

Suppress a deliberate fire-and-forget call with //noinspection StrayCompletableFuture on the line above, or via the usual quick-fix menu.
